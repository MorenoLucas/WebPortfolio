Emigrando Web

Es un blog de viajes realizado con React, donde se sube información y los usuarios se contanctan por medio de comentarios y mensajes por medio de la base de datos de FireBase. La maquetación es hecha con Bootstrap

Tienda de Bolsos

Es un ecommerce realizado con Angular, donde se utuilizo formularios reactivos, servicios para la cesta y pipe, la información se inyecta por medio de base de datos de FireBase. La maquetación fue hecha con Angular Material.

Rover Mars

Proyecto basado en prueba técnica realizado con angular en el bootcamp de Barcelona Activa, se utuilizan pipe, guards y servicios. Maquetación con angular material

Proyecto Chistes

Proyecto que muestra chistes aleatoriamente consumiendo API de web de chistes y API de el tiempo, realizado con HTML/CSS y JAVASCRITP
